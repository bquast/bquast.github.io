Dans ce post, je veux discuter de certaines ressources qui je pense peut être très utile pour obtenir une bonne connaissance des bases de R et Git.


R
----------------------
Un moyen facile de démarrer avec R sans avoir à l'installer est d'utiliser le tutoriel en ligne
[Try R by Code School](http://tryr.codeschool.com/) (en anglais).

Après avoir terminé ce tutoriel, jetez un oeil à la [Datacamp R tutorial](https://www.datacamp.com/courses/introduction-to-r).
Il ya aussi un [cours compagnon]https://www.datacamp.com/courses/data-analysis-and-statistical-inference_mine-cetinkaya-rundel-by-datacamp) à la [Coursera](https://www.coursera.org/) [Data Science](https://www.coursera.org/specialization/jhudatascience/1) spécialisation.

Pour vous avez déjà fait un travail quantitative dans Excel, puis jeter un oeil à [How to Transition from Excel to R](https://districtdatalabs.silvrback.com/intro-to-r-for-microsoft-excel-users).

Si vous voulez commencer à utiliser R pour un travail sérieux, mais ne savez pas comment commencer,
envisager d'utiliser un **G**raphical **U**ser **I**nterface (GUI) pour R.
L'avantage de ceci est que vous pouvez cliquer sur les dialogues, ce qui est un processus facile.
Après quoi, le programme émet alors tout simplement les deux les résultats, ainsi que le code sous-jacent utilisé pour produire ces résultats.
La lecture du code produit par l'interface graphique est un moyen très facile à apprendre de nouvelles méthodes.

Une bonne interface graphique pour la construction des graphiques est [GrapheR](http://cran.r-project.org/web/packages/GrapheR/vignettes/manual_en.pdf).

Une interface graphique de l'objet plus général (il fait aussi la modélisation) est [Rcmdr](http://socserv.mcmaster.ca/jfox/Misc/Rcmdr/).


Git
---------------------------------
Un moyen facile de démarrer avec Git sans avoir à installer à l'aide du tutoriel en ligne
[Try Git by Code School](https://try.github.io/) (noter que ce site est hébergé sur Github pages).

Après avoir terminé ce tutoriel, vous voudrez peut-être jeter un oeil à ce cool [Jeux de Git Branching](http://pcottle.github.io/learnGitBranching/),
qui va vraiment fortifier votre compréhension quel contrôle est et comment l'utiliser Git pour le gérer.


plus de R
----------------------------------------
Sur un jour après jour, vous ne aurez pas besoin de plus que quelques commandes de git.
Une fois que vous aurez maîtrisé les bases,
il ne est pas beaucoup plus à apprendre ce qui sera utile pour vous l'analyse.

Avec R cependant, ce est une autre histoire, R est à la fois beaucoup plus large,
ainsi que constante évolution.
Pour cette raison, je ajoute un peu plus de ressources qui pourraient être utiles pour acquérir une compréhension plus profonde de R.

Le mathamatician célèbre[John D. Cook](http://www.johndcook.com/)a également un guide intitulé
[R programming for those coming from other languages](http://www.johndcook.com/R_language_for_programmers.html).

Une fois que vous êtes devenu très familier avec R et que vous voulez vraiment apprendre certains sujets plus fondamentaux,
alors jetez un oeil au [Advanced R](http://adv-r.had.co.nz/) de [Hadley Wickam](http://had.co.nz/).


RStudio
-----------------------------
Je mentionne cet outil de R dernière, mais ce est de loin l'outil le plus important de R.
[RStudio](http://www.rstudio.com/) est un  **I**ntegrated **D**evelopment **E**nvironment (IDE),
qui est utilisé par presque tous les utilisateurs professionnels de R.
Les développeurs de RStudio sont également très à la pointe du développement de nouveaux packages R, tels knitr, rmarkdown, shiny, etc.


spécialisation Data Science
-----------------------------
Comme mentionné ci-dessus la plate-forme d'apprentissage en ligne [Coursera](https://www.coursera.org/) a une collection de cours appelé [Data Science Specialisation](https://www.coursera.org/specialization/jhudatascience/1).


Les cours sont dispensés par l'Ecole Johns Hopkins du département Santé publique et de biostatistique avoir un relience lourd sur R(Studio).
Sur le plan thématique, les cours sont axés sur des études de cas de santé publique, mais les leçons se traduisent bien d'autres domaines.
